# jsonforms-primevue-monorepo

## Motivation

The JSONForms.io packages include integrations for vanilla HTML and Veutify, but we need PrimeVue.
This first version depends on PrimeVue 3.
(Note PrimeVue 4 has breaking changes.)

## Packages

See the library [package README](./packages/jsonforms-primevue/README.md) for details.
See the [playground](./examples/playground/README.md) for a live example to test your schemas.
